# rock-image-builder
Image builder for radxa Borads

## Current supported Boards:
  - rock 3a
  - rock 4b
  - rock 4se
  - rock 5a
  - rock 5b

# INSTALLATION:
------------------------------------------------------------
     git clone https://github.com/byte4RR4Y/rock-image-builder
     cd rock-image-builder
     chmod +x install.sh
     sudo ./install.sh
------------------------------------------------------------
